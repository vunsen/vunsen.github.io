# Tech Nexus - 科技博客

一个酷炫科技风的 GitHub 博客页面

## 功能特性

- 🎨 **深色科技主题** - 采用深黑色背景配合霓虹色彩
- ✨ **动态粒子背景** - 50个粒子浮动动画效果
- 🔮 **Glitch 文字效果** - 故障艺术风格标题动画
- ⌨️ **打字机效果** - 副标题逐字显示动画
- 💫 **发光按钮** - 渐变光效交互按钮
- 🚀 **卡片悬停效果** - 项目卡片悬停动画
- 🖱️ **光标跟随器** - 鼠标跟随特效
- 📱 **响应式设计** - 完美适配移动端

## 技术栈

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- JavaScript (ES6+)

## 页面结构

```
├── 首页 Hero 区域
├── 关于我部分
├── 项目展示
├── 博客文章
└── 页脚联系信息
```

## 运行方式

```bash
# 克隆仓库
git clone https://github.com/vumsen/vumsen.github.io.git

# 进入目录
cd vumsen.github.io

# 使用任意 HTTP 服务器启动
python -m http.server 8000
# 或
npx serve
```

## 预览

访问 `http://localhost:8000` 查看效果
